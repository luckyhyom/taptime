# Taptime

**탭 한 번으로 시작하는 나만의 시간.**

---

하루를 돌아보면 "오늘 뭐 했지?" 싶은 날이 있습니다. 분명 바빴는데 뭘 했는지 모르겠고, 열심히 한 것 같은데 기록이 없습니다.

Taptime은 그런 하루를 바꿔주는 앱입니다.

공부, 운동, 독서 — 자주 하는 활동을 프리셋으로 만들어두고, 시작할 때 **탭 한 번**이면 됩니다. 끝나면 자동으로 기록되고, 일간/주간/월간 통계로 내 시간이 어디에 쓰이는지 한눈에 보여줍니다.

## 이런 분들을 위해 만들었습니다

- 매일 공부 시간을 기록하고 싶은 학생
- 운동 루틴을 꾸준히 추적하고 싶은 직장인
- "오늘 얼마나 집중했지?" 궁금한 프리랜서
- 시간 관리 앱을 깔았다가 귀찮아서 삭제한 경험이 있는 모든 사람

## 핵심 기능

### 탭 한 번으로 시작

홈 화면에 프리셋 카드가 있습니다. 탭하면 타이머가 시작됩니다. 그게 전부입니다. 카테고리를 고르고, 시간을 설정하고, 프로젝트를 선택하는 번거로움이 없습니다.

### 장소에 도착하면 자동으로

도서관에 등록해두면, 도서관에 도착하는 순간 타이머가 시작됩니다. 떠나면 자동으로 멈춥니다. 폰을 꺼내지 않아도 됩니다.

### 내 시간을 한눈에

오늘 얼마나 집중했는지, 이번 주는 어땠는지, 어떤 활동에 시간을 가장 많이 쓰는지. 히트맵, 차트, 연속 기록으로 보여줍니다. 꾸준함이 눈에 보이면 계속하고 싶어집니다.

### 기록은 남기되, 부담은 없게

타이머를 깜빡했어도 괜찮습니다. 수동으로 과거 활동을 추가할 수 있습니다. 한동안 쓰지 않는 프리셋은 삭제 대신 보관함에 넣어두세요. 기록은 그대로 남습니다.

### 어디서든 이어서

Google 또는 Apple 계정으로 로그인하면 기기 간 동기화가 됩니다. 인터넷이 없어도 앱은 완벽하게 동작하고, 연결되면 자동으로 동기화됩니다.

## 전체 기능 목록

| 기능 | 설명 |
|------|------|
| 프리셋 타이머 | 이름, 시간, 아이콘, 색상을 설정한 커스텀 타이머 |
| 스톱워치 모드 | 시간 제한 없이 활동 추적 |
| 휴식 타이머 | 세션 종료 후 5분/15분 휴식 |
| 자동 세션 저장 | 완료/중단 시 자동 기록 |
| 수동 세션 입력 | 과거 활동을 직접 추가 |
| 일간/주간/월간 통계 | 바 차트, 도넛 차트, 목표 달성률 |
| 프리셋별 히트맵 | GitHub 스타일 활동 시각화 |
| 연속 기록(스트릭) | 목표 달성 연속 일수 |
| 위치 기반 자동 트래킹 | 등록한 장소 도착/출발 시 타이머 자동 시작/정지 |
| 역지오코딩 | 지도 탭 시 장소명 자동 입력 |
| 프리셋 보관 | 삭제 없이 숨기기, 기록 보존 |
| 드래그 앤 드롭 정렬 | 홈 화면 프리셋 배치 |
| 클라우드 동기화 | Supabase 기반, Google/Apple 로그인 |
| 오프라인 퍼스트 | 인터넷 없이 완전 동작, 연결 시 자동 동기화 |
| 다크 모드 | 시스템 설정 연동 또는 수동 전환 |

## 기술 스택

| 영역 | 기술 |
|------|------|
| 프레임워크 | Flutter (iOS + Android) |
| 상태 관리 | Riverpod |
| 로컬 DB | Drift (SQLite) |
| 클라우드 | Supabase (PostgreSQL + Auth) |
| 아키텍처 | 2-layer MVVM + Repository Pattern |
| 위치 서비스 | CLLocationManager (iOS Platform Channel) |
| 지도 | flutter_map + OpenStreetMap |
| 장소 검색 | Kakao Local API |

```bash
flutter pub get
flutter run --dart-define-from-file=.env
```
