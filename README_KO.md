# Taptime

**의미있는 시간을 모아보세요.**

---

시간을 기록하고 싶은 마음은 있지만, 매번 앱을 열고 카테고리를 고르고 시작 버튼을 누르는 건 귀찮습니다. 그래서 대부분의 시간 관리 앱은 설치한 지 일주일이면 잊혀집니다.

Taptime은 **시간 기록을 보조하는 앱**입니다. 직접 관리하라고 강요하지 않습니다. 탭 한 번이면 기록이 시작되고, 장소에 도착하면 알아서 기록됩니다. 최대한 자동으로, 최대한 간단하게.

## Time Collection

하루하루 쌓이는 기록이 모이면, 내 시간이 보이기 시작합니다.

오늘 공부를 얼마나 했는지, 이번 주 운동을 빠뜨리지 않았는지, 한 달간 어디에 시간을 가장 많이 썼는지. Taptime이 자동으로 모아주는 기록들이 나만의 시간 컬렉션이 됩니다.

## 핵심 가치

### 한 번의 탭으로 기록

프리셋 카드를 탭하면 바로 시작됩니다. 공부, 운동, 독서 — 자주 하는 활동을 미리 만들어두면 매번 설정할 필요가 없습니다.

### 위치 기반 자동 기록

도서관, 헬스장, 카페 — 자주 가는 장소를 등록하면 도착할 때 타이머가 자동으로 시작되고, 떠나면 자동으로 멈춥니다. 폰을 꺼내지 않아도 기록이 쌓입니다.

### 자동화 + 간단화

시간 관리의 가장 큰 적은 귀찮음입니다. Taptime은 기록하는 과정 자체를 최소화합니다. 타이머를 깜빡했으면 나중에 수동으로 추가하면 되고, 안 쓰는 프리셋은 보관함에 넣어두면 됩니다.

## 기능

| 기능 | 설명 |
|------|------|
| 프리셋 타이머 | 이름, 시간, 아이콘, 색상을 설정한 커스텀 타이머 |
| 스톱워치 모드 | 시간 제한 없이 활동 추적 |
| 휴식 타이머 | 세션 종료 후 5분/15분 휴식 |
| 자동 세션 저장 | 완료/중단 시 자동 기록 |
| 수동 세션 입력 | 과거 활동을 직접 추가 |
| 일간/주간/월간 통계 | 바 차트, 도넛 차트, 목표 달성률 |
| 프리셋별 히트맵 | GitHub 스타일 활동 시각화 |
| 연속 기록(스트릭) | 목표 달성 연속 일수 |
| 위치 기반 자동 기록 | 등록한 장소 도착/출발 시 타이머 자동 시작/정지 |
| 역지오코딩 | 지도 탭 시 장소명 자동 입력 |
| 프리셋 보관 | 삭제 없이 숨기기, 기록 보존 |
| 클라우드 동기화 | Google/Apple 로그인, 기기 간 동기화 |
| 오프라인 퍼스트 | 인터넷 없이 완전 동작, 연결 시 자동 동기화 |
| 다크 모드 | 시스템 설정 연동 또는 수동 전환 |

## 기술 스택

| 영역 | 기술 |
|------|------|
| 프레임워크 | Flutter (iOS + Android) |
| 상태 관리 | Riverpod |
| 로컬 DB | Drift (SQLite) |
| 클라우드 | Supabase (PostgreSQL + Auth) |
| 아키텍처 | 2-layer MVVM + Repository Pattern |
| 위치 서비스 | CLLocationManager (iOS Platform Channel) |
| 지도 | flutter_map + OpenStreetMap |
| 장소 검색 | Kakao Local API |
